participant browser
participant server

    Note left of browser: User writes note and clicks "Save" (form submit intercepted by JS)

    Note right of browser: JS: e.preventDefault(), create note object {content, date}, push to local notes array, clear input, redrawNotes()

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa (JSON: { "content": "...", "date": "..." }, Content-Type: application/json)
    activate server
    Note right of server: Server adds note to notes array with new Date()
    server-->>browser: 201 Created
    deactivate server

    Note right of browser: Page stays loaded; notes are re-rendered locally (including the new one) via DOM-API, no reload
